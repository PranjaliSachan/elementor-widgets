# Accordion Hero Slider for Elementor

A premium, highly customizable horizontal accordion hero slider inspired by sophisticated editorial designs. Built specifically for Elementor, this widget allows you to create stunning industry-focused sections with smooth transitions and dynamic client logos.

## âœ¨ Features

- **Premium Design**: Sophisticated dark theme with elegant typography (Playfair Display & Inter).
- **Smooth Transitions**: Fluid card expansion and collapse using hardware-accelerated animations.
- **Dynamic Client Logos**: Add 1-5 logos per slide. Supports both image uploads and text-based client names.
- **Auto-Play**: Built-in auto-play with a visible progress bar for better user engagement.
- **Full Customization**: Control titles, categories, descriptions, button links, and industry-specific background gradients directly from the Elementor editor.
- **Responsive**: Optimized for Desktop, Tablet, and Mobile.
